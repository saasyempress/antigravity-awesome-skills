# 00_intake

Purpose:
This folder receives raw inbound content only.

Allowed inputs:
- Transcripts
- Audio recordings
- Video recordings
- Voice notes
- Long-form notes
- External documents

Rules:
- No editing of content inside this folder
- Files are immutable once added
- Files are named with date + source
- No derived content lives here

Downstream:
Files in this folder are read by autorun systems.
No manual processing happens here.
